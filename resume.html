Binar Hari ke 2 : Resume Git

  • Git status akan fatal jika file belum ada tempat penyimpanan git nya
  • Git add . = menambahkan seluruh perubahan di suatu file ke staged changes
  • Git commit = menyimpan perubahan yg telah dilakukan dengan menambahkan deskripsi atau judul perubahan
  • Git push –u origin = mengirimkan perubahan file yang telah dilakukan ke repository saat pertama kali
  • Git branch –a = untuk melihat branch yg ada
  • Git remote show origin = melihat tujuan/alamat git yang dituu oleh git push
  • Git checkout = untuk pindah branch
  • Melakukan Clone Git orang lain dengan mengambil alamat git yg akan di clone( git clone "alamat git")
